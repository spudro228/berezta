# berezta

Терминальный текстовый редактор на C++17 с поддержкой UTF-8, мультикурсора и закреплённых выделений.

## Возможности

- **UTF-8** — полная поддержка многобайтных символов, включая CJK (double-width)
- **Кодировки** — автоопределение UTF-8, Latin-1 (ISO 8859-1), CP1251 (Windows-1251)
- **Мультикурсор** — Ctrl+W (выделить слово/следующее вхождение), Alt+↑↓ (добавить курсор)
- **Закреплённые выделения** — Ctrl+B закрепляет выделенный текст, Ctrl+L открывает панель для быстрого копирования
- **Поиск и замена** — инкрементальный поиск (Ctrl+F), замена (Ctrl+H)
- **Undo/Redo** — Ctrl+Z / Ctrl+Y с объединением последовательного ввода
- **Удаление строк** — Ctrl+D удаляет строку или несколько выделенных строк
- **Git integration** — подсветка commit messages, корректные exit-коды для git
- **JSON** — подсветка синтаксиса, форматирование и валидация для .json файлов
- **Kitty keyboard protocol** — горячие клавиши работают при любой раскладке клавиатуры
- **Режим центрирования** — F2 для комфортного чтения

## Установка

### Из релиза

Скачайте архив со [страницы релизов](https://github.com/spudro228/berezta/releases/latest).

**macOS** (universal: Apple Silicon + Intel):

```bash
tar xzf berezta-v0.3.0-macos-universal.tar.gz
xattr -cr berezta                        # снять блокировку Gatekeeper
sudo cp berezta /usr/local/bin/
```

> macOS блокирует неподписанные бинарники из интернета. Команда `xattr -cr` снимает карантинный атрибут. Альтернатива: Системные настройки → Конфиденциальность и безопасность → «Всё равно открыть».

**Linux** (x86_64):

```bash
tar xzf berezta-v0.3.0-linux-x86_64.tar.gz
sudo cp berezta /usr/local/bin/
```

### Из исходников

Требования: CMake ≥ 3.16, компилятор C++17. Зависимости скачиваются автоматически.

```bash
git clone https://github.com/spudro228/berezta.git
cd berezta
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
sudo cp build/berezta /usr/local/bin/
```

### Без sudo

```bash
mkdir -p ~/.local/bin
cp berezta ~/.local/bin/
```

Добавьте в `~/.bashrc` или `~/.zshrc`:

```bash
export PATH="$HOME/.local/bin:$PATH"
```

Примените: `source ~/.bashrc` (или `source ~/.zshrc`).

### Проверка

```bash
berezta --version  # или просто
berezta myfile.txt
```

## Использование как редактор для Git

```bash
git config --global core.editor "berezta"
```

После этого git будет открывать berezta для:
- Редактирования commit messages (`git commit`)
- Интерактивного rebase (`git rebase -i`)
- Merge messages (`git merge`)

Berezta автоматически определяет git-файлы и включает:
- Подсветку комментариев (строки с `#`) серым
- Предупреждение если тема коммита > 50 символов (красный текст)
- Счётчик длины темы в статус-баре (`subj:N/50`)
- Корректные exit-коды: сохранил и вышел → коммит проходит, вышел без сохранения → коммит отменяется

## Горячие клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| Стрелки | Перемещение курсора |
| Alt+←/→ | По словам |
| Home / End | Начало / конец строки |
| PageUp / PageDown | Прокрутка |

### Выделение

| Клавиша | Действие |
|---------|----------|
| Shift+Стрелки | Расширить выделение |
| Shift+Alt+←/→ | Выделение по словам |
| Ctrl+A | Выделить всё |

### Мультикурсор

| Клавиша | Действие |
|---------|----------|
| Ctrl+W | Выделить слово под курсором / следующее вхождение |
| Alt+↑/↓ | Добавить курсор выше / ниже |
| Escape | Сбросить до одного курсора |

### Редактирование

| Клавиша | Действие |
|---------|----------|
| Ctrl+C | Копировать |
| Ctrl+X | Вырезать |
| Ctrl+V | Вставить |
| Ctrl+D | Удалить строку (или выделенные строки) |
| Ctrl+Z | Отмена |
| Ctrl+Y | Возврат |

### Закреплённые выделения

| Клавиша | Действие |
|---------|----------|
| Ctrl+B | Закрепить текущее выделение |
| Ctrl+L | Фокус на панель закреплений |
| F3 | Показать / скрыть панель |

В панели закреплений:

| Клавиша | Действие |
|---------|----------|
| ↑/↓ | Навигация |
| Enter | Копировать элемент в буфер обмена |
| Delete | Удалить элемент |
| a | Копировать все элементы |
| x | Очистить все |
| Escape | Вернуться к редактированию |

### Поиск

| Клавиша | Действие |
|---------|----------|
| Ctrl+F | Найти |
| Ctrl+H | Найти и заменить |

### Файл и вид

| Клавиша | Действие |
|---------|----------|
| Ctrl+S | Сохранить |
| Ctrl+Q | Выход |
| F1 | Справка |
| F2 | Центрирование текста |

## Архитектура

Проект разделён на два слоя:

- **`src/core/`** — логика редактора без платформенных зависимостей. Полностью покрыта тестами.
  - `Buffer` — хранение текста как вектор строк, файловый I/O через Codec
  - `Selection` — мультикурсор с автоматическим слиянием пересекающихся диапазонов
  - `Document` — связывает Buffer, Selection, Viewport, History
  - `History` — undo/redo через снимки буфера
  - `Codec` — абстракция кодировок (UTF-8, Latin-1, CP1251) с автоопределением
  - `Search` — поиск и навигация по совпадениям
  - `JSON` — токенизация, валидация, форматирование

- **`src/tui/`** — терминальный UI.
  - `App` — главный цикл: ввод → dispatch → render
  - `Input` — маппинг клавиш в команды, Kitty keyboard protocol
  - `Renderer` — отрисовка буфера, подсветка выделений и JSON
  - `Terminal` — raw mode, ANSI escape sequences
