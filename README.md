# beresta

Терминальный текстовый редактор на C++17 с поддержкой UTF-8, мультикурсора и закреплённых выделений.

## Возможности

- **UTF-8** — полная поддержка многобайтных символов, включая CJK (double-width)
- **Кодировки** — автоопределение UTF-8, Latin-1 (ISO 8859-1), CP1251 (Windows-1251)
- **Мультикурсор** — Ctrl+D (выделить слово/следующее вхождение), Alt+↑↓ (добавить курсор)
- **Закреплённые выделения** — Ctrl+B закрепляет выделенный текст, Ctrl+L открывает панель для быстрого копирования
- **Поиск и замена** — инкрементальный поиск (Ctrl+F), замена (Ctrl+H)
- **Undo/Redo** — Ctrl+Z / Ctrl+Y
- **JSON** — подсветка синтаксиса, форматирование и валидация для .json файлов
- **Kitty keyboard protocol** — горячие клавиши работают при любой раскладке клавиатуры (в поддерживающих терминалах)
- **Режим центрирования** — F2 для комфортного чтения

## Сборка

Требования: CMake ≥ 3.16, компилятор C++17.

```bash
cmake -B build
cmake --build build
```

Зависимости (utfcpp, Google Test) скачиваются автоматически.

## Запуск

```bash
./build/beresta              # новый файл
./build/beresta myfile.txt   # открыть файл
```

## Тесты

```bash
cd build && ctest --output-on-failure
```

Запуск конкретного набора тестов:

```bash
cd build && ./beresta_tests --gtest_filter='BufferTest.*'
```

## Горячие клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| Стрелки | Перемещение курсора |
| Alt+←/→ | По словам |
| Home / End | Начало / конец строки |
| PageUp / PageDown | Прокрутка |

### Выделение

| Клавиша | Действие |
|---------|----------|
| Shift+Стрелки | Расширить выделение |
| Ctrl+A | Выделить всё |

### Мультикурсор

| Клавиша | Действие |
|---------|----------|
| Ctrl+D | Выделить слово под курсором / следующее вхождение |
| Alt+↑/↓ | Добавить курсор выше / ниже |
| Escape | Сбросить до одного курсора |

### Редактирование

| Клавиша | Действие |
|---------|----------|
| Ctrl+C | Копировать |
| Ctrl+X | Вырезать |
| Ctrl+V | Вставить |
| Ctrl+Z | Отмена |
| Ctrl+Y | Возврат |

### Закреплённые выделения

| Клавиша | Действие |
|---------|----------|
| Ctrl+B | Закрепить текущее выделение |
| Ctrl+L | Фокус на панель закреплений |
| F3 | Показать / скрыть панель |

В панели закреплений:

| Клавиша | Действие |
|---------|----------|
| ↑/↓ | Навигация |
| Enter | Копировать элемент в буфер обмена |
| Delete | Удалить элемент |
| a | Копировать все элементы |
| x | Очистить все |
| Escape | Вернуться к редактированию |

### Поиск

| Клавиша | Действие |
|---------|----------|
| Ctrl+F | Найти |
| Ctrl+H | Найти и заменить |

### Файл и вид

| Клавиша | Действие |
|---------|----------|
| Ctrl+S | Сохранить |
| Ctrl+Q | Выход |
| F1 | Справка |
| F2 | Центрирование текста |

## Архитектура

Проект разделён на два слоя:

- **`src/core/`** — логика редактора без платформенных зависимостей. Полностью покрыта тестами.
  - `Buffer` — хранение текста как вектор строк, файловый I/O через Codec
  - `Selection` — мультикурсор с автоматическим слиянием пересекающихся диапазонов
  - `Document` — связывает Buffer, Selection, Viewport, History
  - `History` — undo/redo через снимки буфера
  - `Codec` — абстракция кодировок (UTF-8, Latin-1, CP1251) с автоопределением
  - `Search` — поиск и навигация по совпадениям
  - `JSON` — токенизация, валидация, форматирование

- **`src/tui/`** — терминальный UI.
  - `App` — главный цикл: ввод → dispatch → render
  - `Input` — маппинг клавиш в команды, Kitty keyboard protocol
  - `Renderer` — отрисовка буфера, подсветка выделений и JSON
  - `Terminal` — raw mode, ANSI escape sequences
