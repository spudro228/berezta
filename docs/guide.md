# Beresta — руководство пользователя

## Для чего создан beresta

Beresta — это лёгкий терминальный текстовый редактор для тех, кому нужен удобный инструмент прямо в консоли. Он создан как альтернатива nano и micro с акцентом на:

- **Работу с кириллицей** — автоопределение кодировок (UTF-8, CP1251, Latin-1), горячие клавиши работают при русской раскладке
- **Быстрый сбор фрагментов из файла** — закрепляйте нужные куски текста и копируйте их одним нажатием
- **Мультикурсор** — редактирование нескольких мест одновременно, как в VS Code
- **Работу с JSON** — подсветка, форматирование и валидация прямо в редакторе

Beresta не пытается заменить vim или emacs. Это простой редактор, который можно освоить за пять минут.

---

## Быстрый старт

```bash
# Открыть файл
beresta myfile.txt

# Создать новый файл
beresta
```

Редактирование работает сразу — просто печатайте. Никаких режимов ввода, никаких специальных команд для начала работы.

Для сохранения нажмите **Ctrl+S**, для выхода — **Ctrl+Q**. Если забыли горячие клавиши — нажмите **F1**.

## Основы работы

### Навигация

Перемещайтесь стрелками. Для быстрого перемещения:

- **Alt+←/→** — прыжки по словам
- **Home / End** — начало и конец строки
- **PageUp / PageDown** — постраничная прокрутка

### Выделение текста

Зажмите **Shift** и двигайте стрелками — текст будет выделен синим. **Ctrl+A** выделяет весь текст.

### Копирование и вставка

- **Ctrl+C** — скопировать выделенный текст
- **Ctrl+X** — вырезать
- **Ctrl+V** — вставить

Текст копируется в системный буфер обмена — его можно вставить в любое другое приложение.

## Закреплённые выделения

Это главная особенность beresta. Когда вы работаете с большим файлом и вам нужно собрать несколько фрагментов из разных мест:

### Как это работает

1. Выделите нужный фрагмент текста (Shift+стрелки)
2. Нажмите **Ctrl+B** — фрагмент закрепится, появится зелёная подсветка
3. Перейдите к другому месту в файле
4. Выделите ещё один фрагмент, снова **Ctrl+B**
5. Повторяйте сколько нужно

Справа появится панель со списком всех закреплённых фрагментов. Самые свежие — сверху.

### Работа с панелью

- **Ctrl+L** — переключиться на панель
- **↑/↓** — выбрать фрагмент
- **Enter** — скопировать выбранный фрагмент в буфер обмена
- **a** — скопировать все фрагменты (через перенос строки)
- **Delete** — удалить один фрагмент
- **x** — очистить все
- **Escape** — вернуться к редактированию

### Управление панелью

- **F3** — показать или скрыть панель
- Панель появляется автоматически, когда есть закреплённые фрагменты

### Перекрытие выделений

Если вы закрепите фрагмент, а потом выделите и закрепите текст, который пересекается со старым — старое закрепление автоматически удалится. Это позволяет уточнять выделение без ручной очистки.

### Подсветка при редактировании

Закреплённые области подсвечиваются зелёным в тексте. Если вы редактируете текст до или после закреплённой области — подсветка корректно сдвигается. Если вы редактируете текст внутри закреплённой области — подсветка этой области пропадает (текстовый снимок в панели сохраняется).

## Мультикурсор

Мультикурсор позволяет редактировать несколько мест одновременно.

### Выделение одинаковых слов

1. Поставьте курсор на слово
2. Нажмите **Ctrl+D** — слово выделится
3. Нажмите **Ctrl+D** ещё раз — выделится следующее такое же слово
4. Повторяйте для каждого вхождения
5. Начните печатать — текст заменится во всех выделениях одновременно

### Курсоры на нескольких строках

- **Alt+↓** — добавить курсор строкой ниже
- **Alt+↑** — добавить курсор строкой выше

Полезно для одинакового редактирования соседних строк.

### Сброс

Нажмите **Escape** чтобы вернуться к одному курсору. В статус-баре справа внизу отображается количество активных курсоров.

## Поиск и замена

### Поиск

1. Нажмите **Ctrl+F**
2. Введите текст для поиска — совпадения подсвечиваются в реальном времени
3. **Enter** — перейти к следующему совпадению
4. **Escape** — закрыть поиск

Если перед поиском был выделен текст — он автоматически подставится в строку поиска.

### Замена

1. Нажмите **Ctrl+H**
2. Введите текст для поиска, затем текст замены (разделены `|`)
3. **Enter** — заменить все вхождения
4. **Escape** — закрыть

## Работа с JSON

При открытии `.json` файла автоматически включается:

- **Подсветка синтаксиса** — ключи, строки, числа, булевы значения, скобки
- **Валидация** — при сохранении предупредит о невалидном JSON

---

## Кодировки

Beresta автоматически определяет кодировку файла:

- **UTF-8** — по умолчанию, включая BOM
- **CP1251** — Windows-кириллица (определяется эвристически)
- **Latin-1** — ISO 8859-1

Файл сохраняется в той кодировке, в которой был открыт.

---

## Режим центрирования

Нажмите **F2** — текст отцентрируется по ширине ~80 символов. Удобно для чтения на широких мониторах. Повторное нажатие отключает.

---

## Раскладка клавиатуры

Горячие клавиши работают при любой раскладке. В терминалах с поддержкой Kitty keyboard protocol (Kitty, WezTerm, Ghostty) это происходит автоматически через определение физической клавиши. В остальных терминалах работает маппинг ЙЦУКЕН→QWERTY.
